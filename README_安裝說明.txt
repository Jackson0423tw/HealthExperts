健管家 Vivo X300 桌面安裝版

這是 PWA，不需要 Android Studio，也不需要公司 IT 管理員權限。
安裝後可從 Vivo X300 桌面圖示直接進入，不必經過 Gemini。

安裝方法：
1. 將本資料夾上傳到 HTTPS 網頁空間，例如 GitHub Pages、Netlify 或 Cloudflare Pages。
2. 用 Vivo X300 的 Chrome 開啟網址。
3. 點畫面上的「安裝到桌面」。
4. 若沒有出現按鈕，點 Chrome 右上角 ⋮，選「安裝應用程式」或「加到主畫面」。

資料會保存在手機瀏覽器的本機儲存空間；身高、體重、熱量目標、保護模式及飲食紀錄，下次開啟仍會保留。
